# 📺 Netflix - Clone de Seleção de Perfis

Este projeto é uma recriação da icônica interface "Quem está assistindo?" da Netflix. O objetivo foi praticar conceitos fundamentais de desenvolvimento web front-end.

## 🚀 Tecnologias Utilizadas
- **HTML5**: Estruturação semântica da página.
- **CSS3**: Estilização, layout com Flexbox e responsividade.
- **Git & GitHub**: Controle de versão e hospedagem.

## 🔗 Demonstração
Você pode visualizar o projeto online através do link abaixo:
[Clique aqui para acessar a Demo](https://simarasantos.github.io/projeto-netflix/)

## 🧠 Aprendizados
Durante o desenvolvimento, foquei em:
- Alinhamento centralizado com Flexbox (`justify-content` e `align-items`).
- Organização de pastas de assets (imagens).
- Fluxo de trabalho com Git (init, add, commit e push).
